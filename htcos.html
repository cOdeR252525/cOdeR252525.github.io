how to code on scratch

  1. go to "edit your own project
  2. go to "events" and choose which event will start your code
  3. put different blocks and test your code!

examples of blocks

  1. move: choose the number of steps the sprite will move
  2. turn: choose the number of degrees the sprite will turn
  3. pen down/pen up: put the pen down/up
  4. forever: repeat an action forever
  5. say: say something
